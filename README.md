# bilibiliToolChrome bilibili Chrome 小插件

## 功能

1. 开启夜间模式
2. 选中网页中的文字，可以右键直接使用b站搜索
3. 在地址栏输入bs后按下tab，输入内容可以用b站搜索
4. 在b站视频播放页面可以提取封面
